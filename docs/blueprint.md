# **App Name**: OmniCore

## Core Features:

- AI Assistant: Context-aware AI assistant to execute commands, summarize, and generate content based on user data using the Gemini API as a tool.
- Dashboard: Modular drag-and-drop dashboard with customizable widgets and responsive layouts.
- Task Management: Manage tasks, projects, subtasks with dependencies, priorities, and kanban boards.
- Note-taking: Markdown-based notes with bidirectional linking and graph view.
- Internationalization: Full i18n support for Arabic and English, including RTL/LTR direction switching.
- User Authentication: Email/password and OAuth (Google) authentication with user profiles and roles.
- File Management: Upload and manage files, attaching them to tasks and notes.

## Style Guidelines:

- Primary color: Deep Indigo (#4F3A65) to convey intelligence and depth.
- Background color: Light gray (#F0F2F5), nearly desaturated indigo, for a clean and modern backdrop.
- Accent color: Soft Lavender (#A799B7), analogous to Indigo, for interactive elements and subtle highlights.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-focused look. Body font: 'Inter' (sans-serif).
- Code font: 'Source Code Pro' for displaying code snippets.
- Minimalist, line-based icons to maintain a clean and consistent aesthetic.
- Responsive, mobile-first layout with a focus on content hierarchy and intuitive navigation.
- Subtle transitions and animations for a smooth and engaging user experience.
